## [1.1.0] — 2026-06-29

### 新增功能 & 体验优化

#### 艾宾浩斯复习系统 (SM-2 Spaced Repetition)
- **间隔重复算法**：引入 SM-2 算法，根据每次复习结果打分（记住了/模糊/忘记）动态调整下一次的复习周期，用最少回顾次数达成永久记忆。
- **今日待复习横幅**：侧边栏顶部智能展示待复习内容数量横幅，当没有到期内容时自动隐藏。
- **沉浸式复习模式**：侧边栏支持全屏进入复习环境，包含卡片原文、折叠式笔记备注、打分按钮。
- **自适应跳转原文**：点击 `↗ 跳转原文` 自动在新窗口或激活的现有窗口中定位到该网页高亮划线处。
- **复习报告统计**：每日复习完成后自动弹出统计页面，显示完成数量、记忆率以及明天待复习数量。
- **悬停信息气泡 (Tooltip)**：为设置中复习标签加入 SVG 问号图标，悬浮时以小箭头卡片展示 SM-2 策略解析。

#### 滑动抽屉式（Slide-over）设置面板
- 重构设置面板为 `position: fixed` 全屏覆盖设计，支持通过 translateX 属性配合贝塞尔曲线平滑从右侧滑入。
- 引入粘性定位的设置头部（`.settings-header`），包含圆形微动效果的返回 SVG 箭头按钮，提升了在窄侧边栏环境下的视觉纵深。

#### 自定义警示对话框 (Custom Dialog)
- 移除丑陋的系统 `confirm()` 弹框，通过 HTML5 `<dialog>` 自研符合系统主题的微动确认弹框。
- 引入了圆润的危险红色感叹号 Icon、二级微交互缩放反馈和完全伸缩适配的 Flex 按钮排版。

#### 动态标签与独立删除
- 侧边栏的标签芯片过滤栏更新为根据当前搜索过滤结果反向动态统计，解决了以往空标签的问题。
- 收藏卡片上的标签元素集成了独立删除 `×` 按钮，点击即解除当前收藏与该标签的关联。

#### 🛡️ 商店合规优化 (Store Auditing)
- 精简权限申请，将 `host_permissions` 中的 `"<all_urls>"` 宽泛请求彻底移除，以防商店上架漫长的人工审核延误。
- 增加 `webNavigation` 权限用于捕获单页应用 (SPA) 的无刷新路由变化。
- 新增 `chrome.webNavigation.onHistoryStateUpdated` 路由监听机制，利用 3 层递进延时重试机制彻底解决 SPA 框架页面中切换子路由后高亮失效的业界难点。

## [1.0.0] — 2026-06-27

### 项目启动

**需求来源：** 用户访谈（/grill-me 模式），共进行 11 轮问答，完整记录见 [设计决策文档](./design-decisions.md)。

### 新增功能

#### 内容脚本（content.js / content.css）
- **浮动工具栏**：选中文字后页面内自动弹出操作栏，包含「收藏页面」和「高亮划线」按钮及颜色选择器
- **右键菜单支持**：同时支持通过右键菜单「收藏此网页」和「收藏划线」触发操作
- **划线高亮**：选中文字后以彩色背景高亮，支持 5 种预设颜色
- **高亮恢复**：页面加载完成后，通过 XPath + 偏移量自动还原所有历史划线
- **删除交互**：悬停高亮文字时显示删除按钮（×），点击即删，DOM 同步恢复
- **划线备注功能**：悬停高亮段落可点击备注按钮弹出气泡编辑框，添加/修改/删除本条高亮的备注
- **Toast 通知**：所有操作完成后右下角弹出确认提示，2.2s 后自动消失
- **防重复注入**：通过 `window.__markbuddy_initialized` 标志位避免内容脚本重复执行

#### Service Worker（service-worker.js）
- 初始化注册右键菜单，设置侧边面板自动打开行为
- 集中处理全部消息类型（10 种），统一 async/await 错误处理
- 数据操作：书签 CRUD、划线 CRUD、标签同步、设置读写（含批注更新：`UPDATE_HIGHLIGHT_NOTE`）
- 保存划线时自动关联/创建所属页面的书签记录
- 删除书签时级联删除所有关联划线

#### 侧边面板（panel.html / panel.js / panel.css）
- **深/浅色自适应**：CSS 变量 + `prefers-color-scheme`，跟随系统主题无需手动切换
- **收藏列表**：展示 favicon、标题、域名、收藏时间，支持按域名分组卡片视图
- **两层展开**：点击卡片展开该页所有划线，再次点击折叠
- **划线列表与批注同步**：展开卡片显示高亮原文，支持查看、添加和编辑该条高亮的备注，并实时与页面浮动气泡同步
- **实时搜索**：关键词同时匹配标题、URL 和划线内容，匹配词高亮显示
- **标签过滤**：多标签同时选中（OR 逻辑），「全部」按钮重置
- **标签管理**：弹窗添加标签，支持选择已有标签或输入新标签
- **设置面板**：支持迁移高亮色设置，支持切换默认高亮颜色（5 色预设 + 自定义颜色选择器），支持按域名分组开关切换
- **统计栏**：显示当前视图的收藏数量
- **空状态设计**：无收藏时显示引导提示，无搜索结果时显示对应提示

### 技术架构决策

| 决策 | 说明 |
|------|------|
| Manifest V3 | 遵循 Chrome 最新标准，使用 Service Worker 替代 Background Page |
| XPath 序列化 | 用于精确还原划线位置，相比 CSS Selector 可唯一定位文本节点 |
| `chrome.storage.local` | 本地存储，无需登录，隔离性好，上限 10MB |
| `requestAnimationFrame` 分批渲染 | 恢复多条高亮时避免阻塞主线程（每批 10 条） |
| CSS 变量主题系统 | 单文件控制深/浅色切换，无需 JS 干预 |

### 已知限制（v1）

- SPA 路由切换不触发高亮重恢复（需监听 `popstate` / `history` API）
- 页面 DOM 结构大幅变动后，XPath 失效的划线将被跳过（数据保留但不显示）
- 不支持跨设备同步
- 不支持数据导入/导出
