## Phase 5 — 学习探索与高级交互（已完成）

- [x] **商店权限精简**：从 `manifest.json` 移除 `<all_urls>` 宽泛权限，保护隐私并缩短商店审核周期
- [x] **SPA 页面路由自愈高亮**：监听 `chrome.webNavigation.onHistoryStateUpdated`，多层延时自动恢复高亮
- [x] **多重列表排序**：支持最新/最早收藏、最近修改时间、域名 A-Z/Z-A 等排序，并支持本地持久化
- [x] **艾宾浩斯复习系统**：SM-2 遗忘曲线算法、今日待复习横幅、全屏卡片式学习环境、三挡评分及复习完成总结面板
- [x] **抽屉滑出式设置面板**：重构设置面板为 position: fixed 从右侧平滑滑出，带粘性顶部及返回箭头
- [x] **自绘确认弹框**：使用原生 `<dialog>` 配合红色警告 Icon、按钮缩放动效自定义删除确认框
- [x] **动态标签与独立删除**：标签栏依据过滤后数据动态重算，卡片标签集成 inline `×` 独立移除功能
- [x] **复习标签 Hover 提示框**：复习标签新增 SVG 信息图标，悬停展现高保真 SM-2 算法解析 Tooltip

## Phase 6 — 产品定位对齐（找回优先、复习辅助、数据自由）

- [x] **高亮位置轻量自愈**：当页面文本前后发生轻微变化导致旧偏移不准时，基于保存的原文快照在稳定祖先元素内重新定位，并回写新的偏移；找不到匹配时跳过，避免错位高亮。
- [x] **全局快捷键**：支持 `Alt+Shift+M` 打开/关闭侧边栏，`Alt+Shift+H` 将当前选中文本保存为高亮，降低网页阅读捕获成本。
- [x] **高亮级复习准入入口**：每条高亮可以直接加入或移出当前复习标签队列，补齐 SM-2 后台能力与用户操作入口之间的缺口。
- [x] **Markdown 结构化导出**：支持导出全部收藏、当前搜索/标签过滤结果和单个网页，高亮原文、备注、标签、来源 URL 会进入 `.md` 文件，避免数据困在浏览器扩展里。
- [x] **JSON 备份与导入恢复**：设置面板支持完整导出和恢复本地数据，覆盖收藏、高亮、标签、设置、分组和排序偏好，用于灾备而不是笔记整理。
